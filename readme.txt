== Description ==

Adds widget box to sidebar, that includes title, text, website link.

== Installation ==

1. Unzip the files and upload the plugin directory to '/wp-content/plugins/'
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance &rarr; Widgets to begin.

== Frequently Asked Questions ==
= How can I change styles? =
Enter your custom class name in the CSS Classes field. Example: mywidget
Add CSS styles in your theme for .mywidget
